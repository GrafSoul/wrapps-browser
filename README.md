# Browser for app Wrapps

Separate application for viewing video content.

### Installation for development

```
git clone https://github.com/GrafSoul/wrapps-browser.git
cd wrapps-browser

npm install
// or
yarn install

npm start
// or
yarn start

npm build
// or
yarn build
```
